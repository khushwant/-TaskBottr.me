# TaskBottr.me
To run project:

1) Install packages
    npm install
   and Create Images folder in project directory.

2) Run server:
    node .\server1.js

3) Run feature3.htm:
    URL: http://localhost:8081/feature3.htm

4) Choose file and rotate it by clicking on it.

5) Access the image via URL, you get after uploading it.
